## 📌 项目简介

**A_Share_Agent** 是一个基于 RAG（Retrieval-Augmented Generation）架构的 A 股量化决策辅助系统。系统通过以下四层架构实现智能分析：

1. **感知层**：实时采集 A 股全市场行情数据（涨跌幅、成交额等），30 分钟级内存缓存穿透反爬风控
2. **检索增强层（向量 RAG）**：基于 FAISS 向量数据库 + 智谱 embedding-3 模型，对个股新闻进行语义检索，精准定位最相关的舆情片段
3. **可视化层**：基于 Plotly 构建资金势能仪表盘（Gauge）和涨跌幅分布柱状图，直观展示市场情绪
4. **决策层**：将量价数据与 RAG 检索结果注入 LLM（GLM-4-Flash），生成专业级短线博弈研报

## 🧩 核心模块

### 1. `market_tool.py` — 行情数据感知层

- 调用 AKShare `stock_zh_a_spot()` 接口获取全市场实时行情
- 提供涨幅榜 Top 10、跌幅榜 Top 10、单股检索三个 API
- 使用 `@st.cache_data(ttl=1800)` 实现 30 分钟数据缓存，避免频繁请求被封

### 2. `vector_rag.py` — 向量 RAG 核心模块 ⭐

系统的技术核心，实现完整的向量检索增强生成管线：

- **文本分块** (`chunk_text`)：将新闻长文本按 200 字符切块，50 字符重叠窗口防止信息截断
- **向量化** (`get_embedding`)：调用智谱 `embedding-3` 模型将文本编码为高维语义向量
- **索引构建** (`build_faiss_index`)：使用 FAISS `IndexFlatL2` 构建向量索引
- **语义检索** (`search_relevant_chunks`)：基于余弦/L2 距离查找与查询最相关的 Top-K 文本块
- **一键管线** (`rag_retrieve`)：封装完整流程，带降级容错

### 3. `rag_search.py` — 新闻原始数据抓取层

- 调用 AKShare `stock_news_em()` 接口从东方财富抓取个股新闻
- 对股票名称进行清洗（去除 N / *ST / ST 前缀）
- 提取 Top 10 条新闻标题 + 内容，供向量 RAG 模块使用

### 4. `app.py` — Streamlit 主应用

提供四种分析模式，前三种集成 Plotly 动态可视化 + 向量 RAG + LLM 流式推理，第四种为 Agent 自主决策模式：

| 模式 | 功能 | 可视化 | 策略 |
|------|------|--------|------|
| 🚀 涨幅榜异动狙击 | 扫描当日涨幅 Top 10 | 资金势能仪表盘 + 涨幅柱状图 | 识别真龙/杂毛，预测竞价 |
| 💣 跌幅榜恐慌排雷 | 扫描当日跌幅 Top 10 | 恐慌势能仪表盘 + 跌幅柱状图 | 分析暴跌原因，逃命建议 |
| 🔍 单股深度扫描 | 定向分析指定个股 | 目标势能仪表盘 | 资金意图定性，止损建议 |
| 🤖 Agent 自主决策 | 自然语言对话，Agent 自主调工具 | 工具调用流程展示 | ReAct 循环推理 |

### 5. `agent_tools.py` — ReAct Agent 核心 ⭐

基于 Function Calling 实现的 ReAct (Reasoning + Acting) Agent：

- **工具注册** (`TOOL_DEFINITIONS`)：将行情查询、RAG 检索等函数注册为 LLM 可调用的工具
- **工具执行** (`execute_tool`)：LLM 指定调用哪个工具，系统自动执行并返回结果
- **ReAct 循环** (`run_agent`)：LLM 推理 → 调工具 → 看结果 → 再推理，最多 5 轮
- 支持多轮对话历史、工具调用回调

### 6. `portfolio.py` — 模拟持仓管理

- **模拟买入** (`add_position`)：支持加仓，自动计算成本均价
- **模拟卖出** (`remove_position`)：清仓指定股票
- **浮动盈亏计算** (`get_portfolio_with_pnl`)：对接实时行情计算持仓盈亏
- 持仓数据持久化到 JSON 文件

### 7. `agent_brain.py` — CLI 版 Agent（命令行独立运行）

- 提供不依赖 Web UI 的命令行分析能力
- 同样使用向量 RAG 管线
- 适用于定时任务或脚本化调用

## 🛠️ 技术栈

| 类别 | 技术 | 说明 |
|------|------|------|
| 大语言模型 | GLM-4-Flash | 智谱 AI，通过 OpenAI 兼容 API 调用 |
| Embedding | embedding-3 | 智谱文本向量化模型，语义编码 |
| 向量数据库 | FAISS (faiss-cpu) | Facebook 开源向量检索引擎 |
| 数据可视化 | Plotly | 资金势能仪表盘 (Gauge) + 涨跌幅柱状图 |
| 数据接口 | AKShare | A 股行情 + 财经新闻，开源免费 |
| Web 框架 | Streamlit | 快速构建数据应用 |
| 数据处理 | Pandas / NumPy | DataFrame 操作与向量计算 |
| 环境管理 | python-dotenv | API Key 安全管理 |

## 🚀 快速开始

### 前置要求

- Python 3.8+
- 智谱 AI API Key（[申请地址](https://open.bigmodel.cn/)）

### 安装步骤

```bash
# 1. 克隆项目
git clone <repo-url>
cd A_Share_Agent

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置环境变量
# 创建 .env 文件并填入你的 API Key
echo OPENAI_API_KEY="你的智谱API_Key" > .env
echo OPENAI_BASE_URL="https://open.bigmodel.cn/api/paas/v4/" >> .env

# 4. 启动应用
streamlit run app.py
```

### 命令行模式（可选）

```bash
python agent_brain.py
```

## 🎮 功能演示

### 涨幅榜异动狙击

1. 在侧边栏选择 **🚀 涨幅榜异动狙击**
2. 点击 **⚡ 启动涨幅榜扫描**
3. 左侧展示实时涨幅 Top 10 数据表
4. 右侧展示 AI 游资视角的流式研报

### 跌幅榜恐慌排雷

1. 在侧边栏选择 **💣 跌幅榜恐慌排雷**
2. 点击 **💣 启动跌幅榜扫描**
3. AI 分析暴跌原因，给出散户逃命建议

### 单股深度扫描

1. 在侧边栏选择 **🔍 单股深度扫描**
2. 输入股票代码或名称（如：`贵州茅台` 或 `600519`）
3. 查看该股的实时数据 + RAG 舆情档案 + AI 定向研报

### Agent 自主决策 🆕

1. 在侧边栏选择 **🤖 Agent 自主决策**
2. 在聊天框输入自然语言问题（如："帮我看看今天涨幅榜，分析谁是真龙"）
3. Agent 自主决定调用哪些工具，实时展示推理过程
4. 支持多轮对话，上下文连贯
